Detect Resolution - медиа запросы для js, помогает инициализирывать и деиницализировать различный функционал с помощью генерации событий.

Плагин принимает в качестве параметра массив разрешений и при загрузке страницы (document.ready) или ресайзе страницы генерирует события.

Всегда генерируется событие onchangeresolution в объекте Event котороко передается имя этого разрешения.

Также генерируется событие для каждого передано брейк поинта (on/off). Например: передаем настройками такой объект (это настройки по умолчанию) 

			
var option = {
	breakpoints: {

		xsmall: {				
				finish: 767
			},

		small: {
				start: 768,
				finish: 991
			},

		medium: {
				start: 992,
				finish: 1279
			},

		large: {
				start: 1280,
				finish: 1365
			},
			
		xlarge: {
				start: 1366
			}
	}
}

onxsmall сработает при разрешении 0 - 676 onsmall сработает при разрешении 676 - 791 onmedium сработает при разрешении 791 - 1200 onlarge сработает при разрешении > 1200

события offxsmall и т.д. срабатывают при смене разрешения на другой breakpoint с указаного
